### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation Steps

1. Clone the repository
```bash
git clone https://github.com/SurajGupta2k/todo-app.git
cd todo-app
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Start the development server
```bash
npm run dev
# or
yarn dev
```

4. Build for production
```bash
npm run build
# or
yarn build
```

5. Preview production build
```bash
npm run preview
# or
yarn preview
```
